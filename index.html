EduVault Setup

Connect your free Supabase project

✅ 100% Free · No credit card · Ever

① Go to supabase.com → Sign up free (just email)
② Click "New project" → give it any name → set a password → Create
③ Wait ~1 minute for it to set up
④ Go to Project Settings → API → copy the two values below
⑤ Go to Storage → Create a bucket named resources → make it Public
⑥ Go to Authentication → Providers → Email → make sure it's enabled

Project URL Anon / Public Key
